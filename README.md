## ğŸš€ Getting Started

### Prerequisites

Ensure you have the following installed to run Kaiku locally:
- Node.js (v22+)
- Rust (Latest Stable)
- Tauri CLI

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/detair/canis.git
   cd canis
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development environment:
   ```bash
   npm run tauri dev
   ```
